# PuertoTask
Aplicación web progresiva (PWA) de productividad multiusuario que integra hábitos, estudio, finanzas y organización diaria en una sola plataforma. Permite programar tareas con repetición, registrar progreso y analizar estadísticas. Desarrollada con React y Supabase, instalable en móvil sin infraestructura de pago.
